# YouTrend-AI

An end-to-end AI-powered YouTube Trending Analytics platform built with Streamlit, Machine Learning, NLP, SQL, and Power BI.

## Features

- Interactive Dashboard
- AI Trend Prediction
- NLP Sentiment Analysis
- Model Performance Evaluation
- Power BI Analytics
- SQL-based Business Insights

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NLP
- SQL
- Power BI
