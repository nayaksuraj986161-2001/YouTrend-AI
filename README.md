#  YouTrend AI
### AI-Powered YouTube Trending Analytics Platform

An end-to-end AI-powered web application that analyzes YouTube trending videos using **Machine Learning**, **Natural Language Processing (NLP)**, **SQL**, **Power BI**, and **Streamlit**.

The platform enables users to explore YouTube trending data, predict whether a video is likely to trend, perform NLP-based text analysis, and visualize insights through an interactive dashboard.

# Project Overview

YouTrend AI is a complete YouTube Trending Analytics Platform developed to demonstrate the application of Business Analytics, Machine Learning, NLP, SQL, and Interactive Dashboards.

The project transforms raw YouTube trending datasets into meaningful business insights while providing predictive analytics through a user-friendly Streamlit web application.

The application performs:

- Trend Prediction using Machine Learning
- NLP-based Text Analysis
- Interactive Dashboard Visualization
- Business KPI Analysis
- SQL-based Analytical Insights
- Data Cleaning and Feature Engineering

#  Installation

Clone the repository

```bash
git clone https://github.com/nayaksuraj986161-2001/YouTrend-AI.git
```

Move into the project

```bash
cd YouTrend-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run streamlit/app.py
```
